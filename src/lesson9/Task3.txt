        //        есть ли неточность в программе?

первый параметр - массив который копируем,
второй - индекс в изначальном массиве arr начиная с которого берем элементы для копирования,
третий - массив в который копируем,
четвертый - индекс в массиве куда копируем, начиная с которого вставляем элементы
пятый - количество элементов, которые будут скопированы

из массива берутся элементы 4 5 7 и вставляются в новый массив начиная с индекса 2

Arrays.copyOf(arr, 5) копирует массив arr, который изменили c новой длинной 5
если не использовать переменную для хранения нового массива
из метода Arrays.copyOf(arr, 5), то длина массива arr не изменится
